# Final Project

This project is an Emotion Detection web application built using Python and Flask.

The application analyzes user input text and detects emotions such as:
- Joy
- Anger
- Fear
- Disgust
- Sadness

The project uses Watson NLP Emotion Detection API for emotion analysis.

## Features

- Emotion analysis using NLP
- Flask web interface
- REST API endpoint
- Unit testing with unittest
- Error handling for invalid input

## Technologies Used

- Python
- Flask
- Watson NLP API
- HTML
- JavaScript
